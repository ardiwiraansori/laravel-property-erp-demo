# Mini ERP Property Laravel

Mini ERP Property Management demo built with Laravel, inspired by my experience building and migrating ERP systems from CodeIgniter 3 to Laravel.

This project is created as a portfolio demo to showcase fullstack web development skills in PHP, Laravel, JavaScript, and MySQL for property business operations.

## Background

I have 3 years of experience as a Fullstack Web Developer building ERP systems for a property company. The original system was built with PHP, CodeIgniter 3, JavaScript, and MySQL, then conceptually migrated to Laravel.

This repository is a sanitized demo version for portfolio purposes.

## Main Modules

- Executive Dashboard
- Accounting / Journal
- Cashier
- Cash Management
- Kartu Pembantu
- Tax Module
- KPR Disbursement Tracking
- Bank Progress Tracking
- HR Module
- SPK Pembangunan
- Contractor Disbursement
- Property Development Progress

## Tech Stack

- PHP
- Laravel
- JavaScript
- MySQL
- Bootstrap / Tailwind
- REST API
